========================
The Sphinx Notes Project
========================

.. centered:: *Record anything via* `Sphinx`_

The Sphinx Notes Project aims to recording anything (including but not limited
to blog post, notes, music score, paintings and so on) -- via the
excellent documentation generator - `Sphinx`_ and the powerful markup language -
`reStructuredText`_ !

.. note:: You can move to `my personal site`_ to see how I live in Sphinx.

Currently, These projects are growing under umbrella of The Sphinx Notes Project:

:sphinxnotes:`pages` |pages-badge|
    Github Actions for publishing Sphinx HTML output to github pages.

:sphinxnotes:`snippet` |snippet-badge|
    Non-intrusive snippet manager for Sphinx documentation.

    If you live in terminal, read progam output more than reading HTML page and PDF,
    It can help you view, edit, invoke your documentation snippet in fast speed.

:sphinxnotes:`any` |any-badge|
    Sphinx extension for describing and referencing anything.

    The extension is a way of "constructing a datebase in documentation" by
    introducing `Jinja Template`_ in Sphinx documentation.

:sphinxnotes:`lilypond` |lilypond-badge|
    Sphinx extension for embedding `LilyPond`_ scores.

:sphinxnotes:`isso` |isso-badge|
    Sphinx extension for embeding `Isso comments`_ in documents.

:sphinxnotes:`strike` |strike-badge|
    Sphinx extension for HTML strikethrough text support.
