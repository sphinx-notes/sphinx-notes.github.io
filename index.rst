========================
The Sphinx Notes Project
========================

The Sphinx Notes Project helps people taking and publishing notes with Sphinx_.

.. note:: You can move to `my personal site`_ to see how I live in Sphinx.

Currently, These projects are growing under umbrella of The Sphinx Notes Project:

Projects
========


.. grid:: 1 2 2 2
   :gutter: 2

   .. grid-item-card::
      :link: https://sphinx.silverrainz.me/pages

      pages |pages-badge|
      ^^^^^^^^^^^^^^^^^^^

      Github Actions for publishing Sphinx HTML output to github pages.

   .. grid-item-card::
      :link: https://sphinx.silverrainz.me/strike

      strike |strike-badge|
      ^^^^^^^^^^^^^^^^^^^^^

      Sphinx extension for strikethrough text support.

   .. grid-item-card::
      :link: https://sphinx.silverrainz.me/any

      any |any-badge|
      ^^^^^^^^^^^^^^^

      Sphinx extension for describing and referencing anything.

      The extension is a way of "constructing a datebase in documentation" by
      introducing `Jinja Template`_ in Sphinx documentation.

   .. grid-item-card::
      :link: https://sphinx.silverrainz.me/lilypond

      lilypond |lilypond-badge|
      ^^^^^^^^^^^^^^^^^^^^^^^^^

      Sphinx extension for embedding `LilyPond`_ scores.

   .. grid-item-card::
      :link: https://sphinx.silverrainz.me/mock

      mock |mock-badge|
      ^^^^^^^^^^^^^^^^^

      Sphinx extension for hiding unsupported directives and roles without modifying documents.

   .. grid-item-card::
      :link: https://sphinx.silverrainz.me/isso

      isso |isso-badge|
      ^^^^^^^^^^^^^^^^^

      Sphinx extension for embeding `Isso comments`_ in documents.

   .. grid-item-card::
      :link: https://sphinx.silverrainz.me/snippet

      snippet |snippet-badge|
      ^^^^^^^^^^^^^^^^^^^^^^^

      Non-intrusive snippet manager for Sphinx documentation.

      If you live in terminal, read progam output more than reading HTML page and PDF,
      It can help you view, edit, invoke your documentation snippet in fast speed.
